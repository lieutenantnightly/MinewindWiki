# Fox Hunt	

Fox Hunt is an Event that takes place in the server

- Time Span: 15 Minutes
- Find and kill foxes around the city to get on the scoreboard.
- Players can kill each other but players don't lose their scores
- Foxes themselves will drop random items after killing them
- There tend to be huge clumps of foxes that spawn randomly
- Sometimes a boss fox will spawn, which only counts as 1 fox kill
- A rare Fox-Chan may also spawn, which is a white fox that gives 50 points if killed.

## Notes and tips

- Using elytra and projectiles to kill foxes is faster and easier than using melee.
- Foxes will try to fight rabbit-summons, which counts as kills for the player if a rabbit kills a fox.
- The all-time-high record score is held by Ben_2027 with 1416 fox kills in a single event.
